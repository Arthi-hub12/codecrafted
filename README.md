# Arthi's Portfolio Website

A personal portfolio website built from scratch using HTML, CSS, and JavaScript.
Designed to showcase my projects, skills, and a little bit about who I am.

---

##  About Me

Hi! I'm Arthi, a first year IT student passionate about web development.
This portfolio is both a showcase and a learning project —
built step by step as I grow my skills.

---

##  Features

- Responsive layout that works on all screen sizes
- Smooth scrolling navigation
- Hero section with space-themed background
- About section with personal bio and hobby badges
- Skills section with animated progress bars
- Projects section with cards and GitHub links
- Contact section with social links

---

##  Built With

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (no frameworks)
- Google Fonts (Poppins, Montserrat)
